# app.data.pData    

插件配置管理

### pData.fillData()   
数据填充    
```js
/**
* 数据填充
* @param {object} data 初始数据内容
*/
pData.fillData(data)
```

### pData.addPlugin()
添加插件信息    
```js
/**
* 添加插件信息
* @param {*} options 插件配置
* @returns 
*/
pData.addPlugin(options)
```

### pData.delPlugin()
删除一个插件    
```js
/**
* 删除一个插件
* @param {*} id 对象id
* @returns 
*/
pData.delPlugin(id)
```

### pData.getPlugin()
返回一个一个插件  
```js
/**
* 返回一个一个插件
* @param {*} id 对象id
* @returns 
*/
pData.getPlugin(id)
```

### pData.getPluginList()
返回所有插件（数组）    
```js
/**
* 返回所有插件（数组）
* @returns 
*/
pData.getPluginList()
```

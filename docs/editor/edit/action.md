# 事件动作      
项目的**用户交互**主要由：`事件`、`动作`两部分组成。    
- 事件：分为应用事件、元素事件、用户事件  
  - 应用事件，项目启动后自主事件（`应用启动`、`应用定时器`）
  - 元素事件，元素显示后（`定时任务`、`延迟任务`），内部自定义事件
  - 用户事件，主要指的用户操作元素的事件（当前只有一个`点击`事件）  
- 动作：当前系统内置（显示元素、开关元素、数据传递、模块切换、开关弹窗、新开窗口）与用户自定义动作

相关操作在 `动作面板` 、 `交互面板` 中完成
